# Korean Basic Dictionary to Anki Converter

## 1. はじめに
本ツールは、韓国の国立国語院（국립국어원）が提供する「韓国語基礎辞典（한국어기초사전）」のデータ（JSON形式）を、暗記学習アプリ「Anki」で利用可能なフラッシュカード（CSV形式）に自動変換するためのPythonスクリプトおよびテンプレート群です。（現在日本語のみ対応）

複数の意味（多義語）の統合を行い、見やすいレイアウトのHTML構造を含んだデータを生成します。

## 2. フォルダ構成
本リポジトリの基本的なフォルダ構成は以下の通りです。

```text
📁 korean-dict-to-anki/
 ┣ 📂 anki_card_templates/  ... Ankiのデザイン用テンプレート
 ┃ ┣ 📄 front.html        (表面のHTML)
 ┃ ┣ 📄 back.html         (裏面のHTML)
 ┃ ┗ 📄 style.css         (デザイン指定用CSS)
 ┣ 📂 input_json/           ... 【ユーザー準備】ここに辞典のJSONファイルを入れます
 ┣ 📂 output_csv/           ... 【自動生成】ここにAnki用のCSVが出力されます
 ┣ 📄 formatter.py          ... 変換を実行するメインプログラム
 ┗ 📄 README.md             ... 本説明書
```

## 3. 必要な環境
* **Python 3.x:** スクリプトの実行に必要です。（標準ライブラリのみを使用しているため、追加のインストールは不要です）
* **Anki (PC版):** データのインポートとテンプレートの設定に使用します。

## 4. 使い方（セットアップ手順）

### Step 1: 辞書データ（JSON）の配置
1. 国立国語院のオープンAPI等を利用して取得した「韓国語基礎辞典」のJSONデータを準備します。日本語の対訳を含めることを忘れないでください。
2. 取得した `.json` ファイルを、本リポジトリ内の `input_json` フォルダの中に配置します。フォルダ内のすべてのJSONファイルが自動で一括処理されます。

### Step 2: 変換スクリプトの実行
ターミナル（またはコマンドプロンプト）を開き、本リポジトリのディレクトリで以下のコマンドを実行します。

```bash
python formatter.py
```
実行すると、`input_json`内の全JSONファイルが解析され、`output_csv` フォルダ内に `korean_anki_data.csv` が自動生成されます。
※現在のスクリプト仕様では、音声ファイルの実体ダウンロードは行わず、ストリーミング再生用の音声URLをそのままCSVに出力します。

### Step 3: Ankiのフィールド（項目）設定
1. Ankiのメニューから **「ツール」** ＞ **「ノートタイプを管理」** を開きます。
2. **「追加」** ボタンから新しいノートタイプを作成します（例：「韓国語基礎辞典ノート」）。
3. 作成したノートタイプの **「フィールド」** を開き、以下の **9個のフィールド** を順番通りに作成します。
   1. `표제어` (見出し語)
   2. `동형어 번호` (同形語番号)
   3. `품사_한국어` (品詞_韓国語)
   4. `품사_일본어` (品詞_日本語)
   5. `발음` (発音)
   6. `오디오` (音声URL)
   7. `의미_목록` (意味リスト)

### Step 4: Ankiテンプレートの適用
1. 先ほどのノートタイプ管理画面から、対象ノートを選択して **「カード」** ボタンを押します。
2. 本リポジトリの `anki_card_templates` フォルダ内のファイルをテキストエディタで開き、それぞれ対応する箇所にコピー＆ペースト（上書き）します。
   * **表面のテンプレート (Front Template):** `front.html` の内容
   * **裏面のテンプレート (Back Template):** `back.html` の内容
   * **書式 (Styling):** `style.css` の内容
3. 保存して画面を閉じます。

### Step 5: データのインポート
1. Ankiのメイン画面から **「ファイルを読み込む (Import File)」** をクリックします。
2. `output_csv` フォルダ内の `korean_anki_data.csv` を選択します。
3. オプション画面で以下を設定してインポートを実行します。
   * **ノートタイプ:** 先ほど作成した「韓国語基礎辞典ノート」を選択。
   * **既存のノート:** 「既存のノートを更新 (Update existing notes when first field matches)」を選択（※データ更新時に学習履歴を維持するため）。
   * **フィールドマッピング:** ヘッダーが含まれているため自動で割り当てられますが、1番から7番まで正しく対応しているか確認してください。また、タグ (Tags)のフィールドが割り当てられているかもチェックしてください（任意）。

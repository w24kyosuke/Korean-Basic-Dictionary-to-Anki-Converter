# NIKL-krjp-to-Anki-formatter

本ツールは、韓国の国立国語院（NIKL: National Institute of Korean Language）が提供する「韓国語-外国語学習辞典（韓日）」などのJSONデータを、暗記学習アプリ「Anki」にインポートしやすいCSV形式へと自動整形するためのPythonスクリプトです。

同音異義語や多義語の自動マージ、品詞の日本語変換、Ankiカード上で見やすく表示するためのHTMLタグ化などを一括で行うことができます。

---

## 主な機能
* **一括処理**: 指定ディレクトリ内の複数のJSONファイルを一括で読み込み、1つのCSVに出力します。
* **データ結合**: 同一の見出し語と番号（同形語番号）を持つデータ（複数の意味や例文）を1つのレコードに整理・結合します。
* **品詞の日本語化**: 「명사」を「名詞」に、「동사」を「動詞」になど、韓国語の品詞名を自動的に日本語表記に変換します。
* **HTML構造化**: Ankiカード上で意味リストが綺麗に表示されるよう、余分な改行を省きつつ、CSSでスタイリングしやすいHTML構造 (`<div class="sense-item">`等) に変換して出力します。

---

## 動作環境
* Python 3.6 以上
* 標準ライブラリのみを使用しているため、追加のパッケージ（pip install等）は不要です。

---

## 使い方

### 1. フォルダの準備
スクリプト（`formatter.py`）と同じ階層に、以下の2つのディレクトリを作成してください。

* `input_json/` （変換元のJSONファイルを入れるフォルダ）
* `output_csv/` （変換後のCSVファイルが出力されるフォルダ）

### 2. JSONファイルの配置
国立国語院のオープンAPI等から取得した辞書データ（JSON形式）を、先ほど作成した `input_json/` ディレクトリの中にすべて配置します。ファイル名は何でも構いません（拡張子が `.json` であるすべてのファイルが処理対象となります）。

### 3. スクリプトの実行
ターミナル（またはコマンドプロンプト）を開き、本スクリプトのあるディレクトリに移動して、以下のコマンドを実行します。

```bash
python formatter.py
```

### 4. 出力結果の確認
実行が完了すると、`output_csv/` ディレクトリの中に `korean_anki_data.csv` というファイルが生成されます。

#### 【出力されるCSVのカラム（列）構成】
1. **표제어** (見出し語)
2. **동형어 번호** (同形語番号 - 上付き文字用の番号)
3. **품사_한국어** (品詞_韓国語)
4. **품사_일본어** (品詞_日本語)
5. **발음** (発音)
6. **오디오** (オーディオ - 音声ファイルのURL等)
7. **의미_목록** (意味リスト - HTML構造化済みテキスト)
8. **테그** (タグ - 重要度や等級など)

## Ankiへのインポート方法

1. Ankiデスクトップアプリを開き、「ファイルを読み込む (Import)」を選択します。
2. 出力された `korean_anki_data.csv` を選択します。
3. 読み込みオプション画面にて、以下の設定を確認してください。
   * **フィールドの区切り文字**: コンマ (Comma)
   * **HTMLの使用を許可する**: 必ずチェックを入れてください。（`의미_목록` 列にHTMLタグが含まれているため）
4. 使用したいノートタイプ（フィールド構成）に合わせて、CSVの各列がAnkiのどのフィールドにマッピングされるかを設定し、「インポート」をクリックします。

※ CSSスタイリングのヒント:
Ankiのカードテンプレート（スタイル設定）に以下のようなCSSを追加することで、意味リストを綺麗に表示できます。
```css
.sense-item { margin-bottom: 8px; }
.sense-num { font-weight: bold; color: #555; }
.sense-trans { font-size: 1.1em; font-weight: bold; color: #0056b3; }
.sense-def-kr { font-size: 0.9em; color: #333; }
.sense-def-jp { font-size: 0.9em; color: #666; }
```
